Règles de confidentialité pour l'application EasyGPT

Introduction

Nous sommes engagés à protéger la vie privée de nos utilisateurs et à assurer la sécurité de leurs données. Les règles de confidentialité suivantes décrivent comment nous collectons, utilisons et protégeons les données de nos utilisateurs.

Les types de données que nous collectons

Nous collectons les chats des utilisateurs qui sont analysés pour préparer les réponses du chatbot.

La manière dont nous collectons les données

Les données sont collectées lorsque l'utilisateur envoie un chat.

La finalité pour laquelle nous utilisons les données

Nous utilisons les données collectées pour préparer la réponse du chatbot.

Les tiers avec qui nous partageons les données

Les données collectées sont communiquées avec l'API d'OpenAI pour générer les réponses.

La manière dont nous protégeons les données

Nous ne stockons pas les données collectées. Les données sont uniquement utilisées pour générer une réponse et disparaissent à la sortie de l'application.

Les droits des utilisateurs

Les utilisateurs ont le droit de retirer leur consentement à tout moment en cessant d'utiliser l'application.

La manière dont nous gérons les demandes de retrait de données

Les données collectées ne sont pas stockées et sont supprimées automatiquement à la sortie de l'application. Il n'est pas nécessaire de traiter les demandes de retrait de données.
